Task 0: Getting started

For this problem set, you’ll learn blogdown, an R package that enables building websites through Hugo, a web framework. Like shiny, blogdown enable you to use a more complex framework (i.e., Hugo) but through R functions and syntax that you’ve been learning about all semester.

I recommend running this exercise on your personal laptop, mainly so you’ll have this blog site already saved on your own personal computer.

To begin, it would be helpful to familarize yourself with three great resources:

  • blogdown book. This is a long book and should be used as more of a reference than read from beginning to end. Familarize yourself with the chapters in case you have questions when you run this problem set.

  • Amber Thomas’ Making a Website Using Blogdown, Hugo, and GitHub pages.This is a great overview of using blogdown, hugo, and GitHub; however, this post is from December 2016 and may not be 100% up-to-date with recent changes to packages. Therefore, you may have issues if you implement but the general descriptions are a great overview.

  • Garrick Aden-Buie’s A Blogdown New Post Workflow with Github and Netlify

Task 1: Allison Hill’s Blogdown Workshop Slides

You will develop your own blog using Allison Hill’s Blogdown Workshop Slides.

For this example, you will create a blogdown site and do eight items.

  1. Adding a logo

  2. Updating the about page

  3. Enabling emojis

  4. Changing the navigation bar

  5. Customizing the CSS

  6. Adding font awesome icons to site, and to navigation bar

  7. Making a new home page

  8. Moving posts around

You will be graded for 0.5 points for each of the eight items.
